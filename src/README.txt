To test my program:

1. Open your terminal. Open my project folder. Go to "src" folder.

2. Put your input file in the "src" folder.

3. Copy and paste the following instruction to build my project: 

gcc -std=c99 -Wall -Werror -g -o expr project2.c tree.c stack.c table_driven_parsing.c -lm

4. Use instruction in the following format to run my program:

./expr test.txt

where "test.txt" is the name of your input file
